Puntuación

Completa el fichero script.js para permitir al usuario introducir su puntuación. Para rellenar las estrellas, puedes hacer uso de la clase fa-solid fa-star de font-awesome. Puedes encontrar más información aquí.

También puedes usar la clase text-warning de bootstrap para que se muetren en amarillo. Encuentra más información sobre colores de bootrstrap aquí.

Por último, modifica el div con id puntuacion para que se muestre solo cuando se haya seleccionado una puntuación, y muestre el número de estrellas seleccionadas.
